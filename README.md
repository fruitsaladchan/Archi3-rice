- OS - Arch :3
- wm: i3wm
- bar - Polybar
- browser: ungoogled chromium
- shell: Zsh + oh-my-zsh
- terminal: kitty
- editor: nvim
- notifications: Dunst
- compositor: [pijulius picom](https://github.com/pijulius/picom)
- app launcher: Rofi
- system monitor: [btop](https://github.com/aristocratos/btop)
